## 📖 安装说明

- **Windows**：双击 exe 安装。若 SmartScreen 提示"未知发布者"，点 **更多信息 → 仍要运行**。
- **macOS**：拖入"应用程序"后，首次打开右键点图标选 **打开**（绕过 Gatekeeper）。
- **Android**：允许"未知来源"安装。新机装 arm64 版，老机/不确定装通用版。

## ✨ 功能

- 内核 [mihomo](https://github.com/MetaCubeX/mihomo)，支持 Hysteria2 / AnyTLS / SS / Trojan
- 系统代理 + TUN 全局模式
- 在线支付、套餐购买、签到、邀请返现、在线客服
